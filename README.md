🌟 Wallify – Your Ultimate Wallpaper App 🎨
Wallify is a cross-platform wallpaper app built with React Native, Node.js, Express, and MySQL. It offers a vast collection of high-quality wallpapers, allowing users to browse, download, and set wallpapers effortlessly.

🚀 Features
✅ Stunning collection of HD & 4K wallpapers
✅ User-friendly and intuitive UI
✅ Fast and optimized API with Node.js & Express
✅ Secure and efficient MySQL database for storing wallpapers
✅ Cross-platform support (Android & iOS)

🛠 Tech Stack
Frontend: React Native
Backend: Node.js, Express
Database: MySQL
